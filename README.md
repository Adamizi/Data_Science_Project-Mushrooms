# Data-Science-Projekt: Pilze

Gruppenprojekt zur Klassifikation von Pilzen als essbar oder giftig.

## Dateien

- `data/raw/mushroom_classification_data.csv`: ursprüngliche Datendatei
- `data/processed/mushroom_classification_clean.csv`: bereinigte Datendatei
- `notebooks/mushroom_classification_project.ipynb`: Notebook mit Analyse und Modell
- `report.md`: kurzer Abschlussbericht
- `requirements.txt`: benötigte Python-Pakete

## Kurzbeschreibung

Die Lösung orientiert sich am Unterrichtsnotebook `03_supervised_classification.ipynb`. Es werden ein Entscheidungsbaum und KNN verwendet, um die Zielvariable `poison` vorherzusagen.
